# べんきょうタイマー

6歳の子供の勉強時間を計測するためのアナログ時計付きタイマーアプリです。

## 機能

- **アナログ時計**: 現在時刻をリアルタイム表示。外側に分目盛り（5〜60）、内側に分目盛りと時間数字（1〜12）を表示し、長針・秒針がどこを指しているか見やすい設計
- **べんきょうタイマー**: 「はじめる」「おわる」ボタンで勉強の開始・終了時刻を記録し、かかった時間（分）を表示
- **画面ロック防止**: 計測中は画面が自動ロックされない。「おわる」を押した後も1分間はロックを防止。最大20分を上限とする

> ⚠️ 画面ロック防止機能（Wake Lock API）はHTTPS環境でのみ動作します。`file://` で開いた場合は機能しません。

## 使い方

1. `index.html` をブラウザで開く
2. 勉強を始めるときに「はじめる」をタップ
3. 勉強が終わったら「おわる」をタップ
4. かかった時間が表示される

## 技術

- HTML / CSS / JavaScript のみ（ライブラリ不使用）
- フォント: [M PLUS Rounded 1c](https://fonts.google.com/specimen/M+PLUS+Rounded+1c)
